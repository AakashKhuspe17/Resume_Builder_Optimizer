# 🚀 Resume Analyzer & Job Recommendation System

A smart web application that analyzes resumes and provides personalized job recommendations, skill suggestions, and learning resources using NLP techniques.

---

## 📌 Overview

This project allows users to upload their resume (PDF) and get:
- Extracted personal and professional details  
- Skill-based job role prediction  
- Recommended skills to improve employability  
- Course suggestions for upskilling  
- Resume scoring with improvement tips  

It also includes an **Admin Dashboard** to manage and view user data.

---

## ✨ Features

- 📄 Resume Upload & Parsing (PDF)
- 🧠 NLP-based Skill Extraction
- 🎯 Job Role Prediction (DS, Web Dev, Android, etc.)
- 💡 Smart Skill Recommendations
- 🎓 Course Recommendations
- 📊 Resume Score with Suggestions
- 🎥 Resume & Interview Preparation Videos
- 🗄️ MySQL Database Integration
- 👨‍💻 Admin Panel (View/Delete Data)

---

## 🛠️ Tech Stack

- **Frontend/UI:** Streamlit  
- **Backend:** Python  
- **Libraries:** NLTK, SpaCy, Pandas  
- **Resume Parsing:** PyResParser, PDFMiner  
- **Database:** MySQL  
- **Visualization:** Plotly
